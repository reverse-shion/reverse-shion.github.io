# 詩韻式 星導日リーディング

## 概要

このツールは、タロットと星の対応から、恋愛・復縁・連絡タイミングにおける「動きやすい日」「準備日」「慎重日」を出すための内部鑑定補助コンソールです。

相談者へ未来を断定するためのものではなく、鑑定者が入力したカード情報をもとに、現実的で優しい行動指針へ整えることを目的としています。

## 基準表

星導日リーディングでは、デカン・Court・Major対応は、`assets/js/seido-date.js` の `ANTHONY_LEWIS_DECAN_TABLE` に定義したアンソニー・ルイス式デカン対応表を基準とします。

他体系のデカン対応表とは混ぜません。小アルカナ2〜10、黄道十二宮、Court、Major、デカン開始日・終了日は、この表を正本として扱います。

## v0.4の基本構造

星導日は、内部的に次の要素を組み合わせて計算します。

```text
星導日 = Z + D + N + S + C
```

- `Z`：黄道十二宮
- `D`：デカン範囲
- `N`：日の鍵カードの数字
- `S`：スート補正
- `C`：Court補助

ただし、顧客向けには式や内部点数は見せません。表示するのは次の内容です。

- 星が導く本命日
- 準備日
- 慎重日
- おすすめ行動
- 避ける行動
- 言葉の温度感

## 各カードの役割

- デカンカード：小アルカナ2〜10を基準に、対応する10日間のデカン範囲を決めます。
- 日の鍵カード：デカン範囲内の何日目かを決めます。
- スート：日付補正と行動の質を補います。
- Court：日付決定の主役ではなく、行動スタイル・人物傾向を読む補助です。
- Major：星座全体の心理テーマを読みます。
- Re：プレミアム検証で、その日への執着・固定化リスクを見ます。

## 重要ルール

Courtは日付決定の主役にしません。

Courtは「いつ」ではなく「どう動くか」を読むために使います。日付インデックスは補助的に扱い、鑑定文では行動スタイルや人物傾向として表現します。

## 入力UI v0.4

日の鍵カードは、タブ式で選択します。

### 大アルカナ

大アルカナタブでは、カード名を選ぶだけで計算用番号が自動設定されます。

内部入力は次の形式です。

```text
day_key_card_type = 'major'
day_key_card_suit = 'Major'
day_key_card_rank = 'Major'
day_key_card_number = 選択した大アルカナ番号（0〜21）
```

Major番号は手入力しません。計算時は既存ロジックにより1〜10へ正規化されます。

### 小アルカナ

小アルカナタブでは、スートとランクを選びます。

スートは `Wands` / `Cups` / `Swords` / `Pentacles`、ランクは `Ace` / `2`〜`10` / `Page` / `Knight` / `Queen` / `King` です。

内部入力は次の形式です。

- Ace：`day_key_card_type = 'ace'`
- 2〜10：`day_key_card_type = 'numbered'`
- Page / Knight / Queen / King：`day_key_card_type = 'court'`

小アルカナでは `day_key_card_number` は空欄です。日付インデックスはランクから自動計算します。

```text
Ace = 1
2〜10 = その数字
Page = 2
Knight = 4
Queen = 7
King = 10
```
