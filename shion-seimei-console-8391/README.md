# 詩韻式 星命未来鑑定コンソール

## 概要

このツールは、算命学の考え方を参考にした「詩韻式 星命鑑定」の内部用鑑定補助コンソールです。生年月日から算命学参考データを表示し、詩韻式の星命タイプとタロット共鳴カードを組み合わせて、相談者へ渡しやすい鑑定文の下書きを作成します。

Phase 1では、自己理解型の鑑定に加えて「今年の総合運」「人生の転換期」「月別運勢」「恋愛・出会いが動きやすい時期」「仕事・金運が動きやすい時期」「注意すべき月」「開運アクション」を表示します。

## 注意

- 算命学の厳密鑑定を完全再現するものではありません。
- MVPでは節入り計算、大運、天中殺、十大主星、十二大従星は断定しません。
- 未来鑑定スコアは厳密な算命学の大運・月運・天中殺ではありません。
- 現段階では「体感的な転換期」「動きやすい月」として柔らかく扱います。
- 月柱は節入り未完全対応のため「簡易月柱」と表示します。
- タロットとの関係は伝統的対応ではなく、詩韻式の共鳴カードです。
- 顧客情報を保存しません。
- 入力内容を外部送信しません。
- localStorage、sessionStorage、Cookie、外部API、DBは使用していません。
- 簡易パスワードは完全なセキュリティではありません。
- GitHub Pages上で本格的な顧客情報管理をしない前提です。
- 本格運用では private repository、認証付きホスティング、またはローカル運用を検討してください。
- 医療、法律、投資、借金、破産、妊娠、寿命などを断定する鑑定には使わないでください。

## Phase 1で追加された機能

- 鑑定対象年の入力欄を追加しました。未入力の場合は現在年にフォールバックします。
- 78枚タロットデータを内部データとして扱えるようにしました。
- 1月〜12月の未来鑑定スコアを deterministic に生成します。
- 今年の総合運、転換期、月別運勢、恋愛・出会い運、仕事・金運、注意時期、開運アクションを鑑定文へ統合します。
- タロット未選択でも鑑定文生成ができるようにしました。
- 未来鑑定セクションのコピーと、最終鑑定文全文コピーに対応しました。

## 使い方

1. ページにアクセス
2. パスワード入力（初期パスワード: `shion-star-console`）

6. タロット共鳴を反映

8. セクションまたは最終鑑定文をコピー

## 算命学参考データの計算方針

### 年柱

- 60干支の年サイクルを使う簡易計算です。
- 西暦4年を甲子年として扱います。
- MVPでは立春を2月4日固定とし、誕生日が2月4日前の場合は前年扱いにします。
- 表示は「簡易年柱・要確認」です。

### 月柱

- 節月ベースの近似境界テーブルを使います。
- 寅月2/4頃、卯月3/6頃、辰月4/5頃、巳月5/6頃、午月6/6頃、未月7/7頃、申月8/8頃、酉月9/8頃、戌月10/8頃、亥月11/7頃、子月12/7頃、丑月1/6頃として扱います。
- 月干は年干から寅月の月干を決めて順送りします。
- 厳密な節入り時刻は未対応のため「簡易月柱」と表示します。

### 日柱

- Julian Day Number を用いた簡易60干支計算です。
- 実装式: Gregorian calendar のJDNを算出し、`(対象日JDN - 基準JDN) mod 60` で干支を求めます。
- 基準: `1984-02-02` を甲子日として扱う簡易基準（JDN `2445703`）です。
- この基準式はMVP用のため、厳密な暦法データによる検証が必要です。
- 表示は「簡易日柱・要確認」です。

## 78枚タロット対応の考え方

Phase 1では `js/tarot-78.js` に78枚すべてのカードを定義しています。既存の `js/tarot-mapping.js` は後方互換のため維持し、既存の `ShionTarotMapping.TAROT_CARDS` と `findTarotByName` は動き続けます。

分類は次の4種類です。

- `major`: 大アルカナ22枚。人生テーマ、魂の課題、大きな転換期を読みます。
- `numbered`: 小アルカナ数札36枚。2〜10 × 4スートで、月別運勢や日常の流れを読みます。
- `ace`: エース4枚。新しいチャンス、始まり、きっかけを読みます。
- `court`: コートカード16枚。人物像、出会い、相手の傾向、関係性を読みます。

Phase 1では逆位置フラグは実装せず、各カードの `shadow` と `caution` で影や注意点を扱います。

## 未来鑑定スコア

`js/future-score.js` は鑑定対象年の1月〜12月に対して、恋愛、仕事、金運、人間関係、注意、転換期のスコアを0〜100の整数で生成します。

スコアは次の情報を簡易的に組み合わせます。

- 生年月日と鑑定対象年から作る deterministic なシード
- 既存の星命タイプと五行バランス
- 相談ジャンル
- 選択タロットとの親和性

同じ入力なら同じ結果になります。`Math.random()` は使用していません。これはPhase 1のMVP簡易判定であり、厳密な大運・月運・天中殺・節入り計算ではありません。

## 星命タイプ判定の仕様

1. 日干から基本タイプを決めます。
2. 日干から補助タイプを決めます。
3. 五行バランスの最強五行でニュアンスを補正します。
4. 相談ジャンルで鑑定文の焦点を変えます。
5. 判定根拠を画面に表示します。

星命タイプは算命学そのものではなく、詩韻式独自解釈です。

## 禁止表現フィルター

鑑定文生成と「詩韻風に整える」処理で、医療・法律・投資・借金・破産・寿命などを断定する表現や、過度に不安を煽る表現を柔らかく現実的な表現へ置換します。
