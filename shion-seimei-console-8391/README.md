# 詩韻式 星命鑑定コンソール

## 概要

このツールは、算命学の考え方を参考にした「詩韻式 星命鑑定」の内部用鑑定補助コンソールです。生年月日から算命学参考データを表示し、詩韻式の星命タイプとタロット共鳴カードを組み合わせて、鑑定文の下書きを作成します。

## 注意

- 算命学の厳密鑑定を完全再現するものではありません。
- MVPでは節入り計算、大運、天中殺、十大主星、十二大従星は断定しません。
- 月柱は節入り未完全対応のため「簡易月柱」と表示します。
- タロットとの関係は伝統的対応ではなく、詩韻式の共鳴カードです。
- 顧客情報を保存しません。
- 簡易パスワードは完全なセキュリティではありません。
- 本格運用では private repository、認証付きホスティング、またはローカル運用を検討してください。
- 医療、法律、投資、借金、破産、妊娠、寿命などの判断には使わないでください。

## 使い方

1. ページにアクセス
2. パスワード入力（初期パスワード: `shion-star-console`）
3. 名前、生年月日、相談ジャンル、悩みメモを入力
4. 基本カルテ生成
5. タロットカードを選択
6. タロット統合ドラフト生成
7. 必要に応じて手編集
8. 鑑定文をコピー

## 実装範囲

- `/shion-seimei-console-8391/` 内で完結するHTML/CSS/JavaScript実装です。
- localStorage、sessionStorage、Cookie、外部API、DBは使用していません。
- `index.html` には `noindex, nofollow` を設定しています。
- 画面内に簡易ロックと個人情報取り扱い注意を表示しています。

## 算命学参考データの計算方針

### 年柱

- 60干支の年サイクルを使う簡易計算です。
- 西暦4年を甲子年として扱います。
- MVPでは立春を2月4日固定とし、誕生日が2月4日前の場合は前年扱いにします。
- 表示は「簡易年柱・要確認」です。

### 月柱

- 節月ベースの近似境界テーブルを使います。
- 寅月2/4頃、卯月3/6頃、辰月4/5頃、巳月5/6頃、午月6/6頃、未月7/7頃、申月8/8頃、酉月9/8頃、戌月10/8頃、亥月11/7頃、子月12/7頃、丑月1/6頃として扱います。
- 月干は年干から寅月の月干を決めて順送りします。
- 厳密な節入り時刻は未対応のため「簡易月柱」と表示します。

### 日柱

- Julian Day Number を用いた簡易60干支計算です。
- 実装式: Gregorian calendar のJDNを算出し、`(対象日JDN - 基準JDN) mod 60` で干支を求めます。
- 基準: `1984-02-02` を甲子日として扱う簡易基準（JDN `2445703`）です。
- この基準式はMVP用のため、厳密な暦法データによる検証が必要です。
- 表示は「簡易日柱・要確認」です。

### 検証サンプル

- `1984-01-03` は立春前のため、簡易年柱では前年扱いになります。
- `1984-02-04` 以後は簡易年柱で当年扱いになります。
- 日柱はJDN基準式に依存するため、専門暦表との照合が今後のTODOです。

## ステータスラベル

- `verified`: 検証済み
- `simplified`: 簡易判定
- `needsReview`: 要確認
- `notImplemented`: 未実装

## タロット共鳴カードの仕様

- タロットは算命学結果を決定しません。
- タロットは今の状態、課題、行動指針を見るために使います。
- 星命タイプとの関係は「対応」ではなく「共鳴」と表現します。
- 1枚引き、3枚引き、5枚引きに対応しています。

## 星命タイプ判定の仕様

1. 日干から基本タイプを決めます。
2. 日干から補助タイプを決めます。
3. 五行バランスの最強五行でニュアンスを補正します。
4. 相談ジャンルで鑑定文の焦点を変えます。
5. 判定根拠を画面に表示します。

星命タイプは算命学そのものではなく、詩韻式独自解釈です。

## 禁止表現フィルター

鑑定文生成と「詩韻風に整える」処理で、医療・法律・投資・借金・破産・寿命などを断定する表現や、過度に不安を煽る表現を柔らかく現実的な表現へ置換します。
