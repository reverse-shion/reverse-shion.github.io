# 詩韻 -シオン- リポジトリ作業指針

- このリポジトリは「詩韻 -シオン-」ブランドの占い・星命カード関連サイトである。
- 既存ページを壊さない。特に既存の `/seimei-card/` は保護する。
- HTML/CSS/JSのみで実装する。
- 外部API、DB、保存機能を使わない。
- 算命学の未検証ロジックを断定しない。
- 「算命学参考データ」と「詩韻式独自解釈」を分離する。
- タロット連動は「対応」ではなく「共鳴カード」と表現する。
- 鑑定文では医療・法律・投資・借金・破産・妊娠・寿命などを断定しない。
- 出力文は怖がらせず、現実的で優しく、行動指針を明確にする。
